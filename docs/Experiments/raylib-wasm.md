# Raylib wasm

## Description
Running raylib project in browser compiled as webassembly

See project at:

<https://github.com/protoni/raylib-stuff>

## Setup
```powershell
docker build -t raylib-wasm .
```

## Run
```powershell
docker run -p 8080:8080 raylib-wasm
```

## View website
```powershell
http://127.0.0.1:8080
```
